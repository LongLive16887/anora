# Быстрый старт

## Шаг 1: Получение модели YOLO 8n

### Вариант A: Использование Python скрипта (рекомендуется)

1. Установите зависимости:
```bash
pip install ultralytics
```

2. Запустите скрипт конвертации:
```bash
python convert_model.py
```

3. Переместите созданный файл `yolov8n.tflite` в:
```
app/src/main/assets/yolov8n.tflite
```

### Вариант B: Ручная конвертация

1. Скачайте модель YOLOv8n:
```python
from ultralytics import YOLO
model = YOLO('yolov8n.pt')
model.export(format='tflite', imgsz=640)
```

2. Переместите `yolov8n.tflite` в `app/src/main/assets/`

## Шаг 2: Открытие проекта

1. Откройте Android Studio
2. Выберите "Open an Existing Project"
3. Укажите папку `comp-vision-app`
4. Дождитесь синхронизации Gradle

## Шаг 3: Запуск приложения

1. Подключите Android устройство или запустите эмулятор
2. Нажмите "Run" (Shift+F10) или кликните на зеленую стрелку
3. При первом запуске разрешите доступ к камере
4. Наведите камеру на объекты - они будут автоматически распознаваться!

## Возможные проблемы

### Ошибка: "Model file not found"
- Убедитесь, что файл `yolov8n.tflite` находится в `app/src/main/assets/`
- Проверьте имя файла (должно быть точно `yolov8n.tflite`)

### Ошибка: "Camera permission denied"
- Перейдите в Настройки → Приложения → Object Detection → Разрешения
- Включите доступ к камере

### Приложение не запускается
- Проверьте, что минимальная версия Android SDK 24
- Убедитесь, что устройство/эмулятор поддерживает камеру

## Настройка параметров

В файле `YOLODetector.kt` можно изменить:

- `confidenceThreshold` (по умолчанию 0.5) - порог уверенности детекции
- `nmsThreshold` (по умолчанию 0.4) - порог для Non-Maximum Suppression
- `modelInputSize` (по умолчанию 640) - размер входного изображения

## Производительность

- Для лучшей производительности используйте устройство с поддержкой GPU
- Можно включить GPU ускорение в TensorFlow Lite (требует дополнительной настройки)
- На слабых устройствах можно уменьшить `modelInputSize` до 416 или 320
